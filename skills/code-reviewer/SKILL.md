# Code Reviewer Skill

Автоматический review кода: поиск багов, стилистических проблем, уязвимостей.

## Использование

Покажите агенту код или путь к файлу и скажите "проверь код" / "code review".

## Что проверяет

- Syntax errors
- Потенциальные баги (None checks, off-by-one)
- Стиль (PEP8 для Python, общие рекомендации)
- Безопасность (SQL injection, XSS, hardcoded secrets)
- Производительность (лишние циклы, N+1)
